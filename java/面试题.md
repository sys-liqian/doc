#### CAP的概念

C：一致性，在分布式系中的所有数据备份，在同一时刻是否同样的值。（等同于所有节点访问同一份最新的数据副本）

A：可用性 ，保证每个请求不管成功或者失败都有响应

P：分区容错性，系统中任意信息的丢失或失败不会影响系统的继续运作
